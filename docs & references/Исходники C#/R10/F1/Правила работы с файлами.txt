Иначе при дозаипе, будет наслоение данных на последнюю строку с данными

Пример:

SPFB.Si,1,20110111,100200,30725.00000,30735.00000,30692.00000,30735.00000
SPFB.Si,1,20110111,100300,30731.00000,30731.00000,30704.00000,30715.00000
(тут должен стоять пустой указатель)
